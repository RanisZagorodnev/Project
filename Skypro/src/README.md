# SkyPro Project

## Цель проекта

Проект предназначен для практики работы с Python и обработки данных.  
Основные функции:  

- Маскировка номеров карт и счетов (`masks.py`)  
- Работа с датами и форматирование (`widget.py`)  
- Фильтрация и сортировка списков словарей (`processing.py`)

## Тестирование

Используемые библиотеки

- Pytest
- Pytest --cov=src

Описание написанных тестов

- Тесты маскировки: Проверка корректности скрытия цифр для карт разных платежных систем и счетов разной длины.
- Тесты виджета: Параметризованные тесты, проверяющие устойчивость функции к разным типам входных строк и корректность парсинга дат.
- Тесты обработки: Проверка логики фильтрации списков и корректности сортировки транзакций (от более ранних к более поздним).
- Граничные случаи: Тестирование поведения функций при получении пустых строк или некорректных форматов данных.

Для запуска всех тестов выполните команду в терминале:
pytest

Для проверки того, насколько полно тесты проверяют исходный код, используется плагин cov.
pytest --cov=src
